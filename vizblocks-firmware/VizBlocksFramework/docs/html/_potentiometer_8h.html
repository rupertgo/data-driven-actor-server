Detailed Description

Slider and rotary potentiometer functionality.

This file contains a class for a psuedo-event-driven potentiometer input. It is inspired by the event-driven logic of the Button class, but analogue pins on the esp8266 cannot have interrupts attached to them so it polls the value of the pin, rather than a change flag. The readings are filtered to improve stability following this norwegian creations tutorial.
